# Используем SCSS

## Преимущества перед CSS

SCSS — "диалект" языка SASS. А что такое SASS? SASS это язык похожий на HAML, но предназначенный для упрощения создания CSS-кода. Рассмотрим несколько преимуществ 
этого диалекта:
